# Gerador de Etiquetas para Pets

Sistema web para geração e impressão de etiquetas personalizadas para animais de estimação.

## 🚀 Deploy Online (Opções Rápidas)

### Opção 1: Netlify (RECOMENDADO - Mais Fácil)

1. Acesse [netlify.com](https://netlify.com)
2. Clique em **Sign Up** e faça login com GitHub
3. Clique em **Add new site** → **Import an existing project**
4. Selecione seu repositório `etiquetas-pet`
5. Configure:
   - **Build command:** `npm install`
   - **Publish directory:** `.` (ou deixe vazio)
6. Clique em **Deploy**
7. Seu site estará online em poucos segundos!

A URL será algo como: `https://etiquetas-pet.netlify.app`

### Opção 2: GitHub Pages (Grátis)

1. Vá para seu repositório no GitHub
2. Clique em **Settings** → **Pages**
3. Em "Source", selecione **Deploy from a branch**
4. Selecione branch **master** e pasta **/ (root)**
5. Clique **Save**
6. Seu site estará online em: `https://seu-usuario.github.io/etiquetas-pet`

### Opção 3: Vercel

1. Acesse [vercel.com](https://vercel.com)
2. Clique em **Sign Up** com GitHub
3. Clique em **Import Project**
4. Selecione `etiquetas-pet`
5. Clique **Deploy**

## 💻 Usar Localmente

```bash
git clone https://github.com/denysapetit-glitch/etiquetas-pet.git
cd etiquetas-pet
npm install
# Abra index.html no navegador
```
